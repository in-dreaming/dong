1. dir="ltr" 元素 (绿色背景 - :dir(ltr) 匹配)

2. dir="rtl" 元素 (粉色背景 - :dir(rtl) 匹配)

3. 无 dir 属性的元素 (继承父元素 LTR - 绿色背景)

4. dir="auto" 以 LTR 文本开头 (检测为 LTR - 绿色背景)
