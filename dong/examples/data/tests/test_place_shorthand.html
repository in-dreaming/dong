Test Summary

This test demonstrates the CSS place shorthand properties:

  • place-items - shorthand for align-items and justify-items
  • place-content - shorthand for align-content and justify-content
  • place-self - shorthand for align-self and justify-self

Parsing rules:

  • Single value: applies to both directions
  • Two values: first value for align-*, second value for justify-*
