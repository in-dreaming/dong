CSS image-rendering 属性测试
测试 image-rendering 属性的实现效果：
  • auto - 默认双线性过滤（平滑缩放）
  • pixelated - 最近邻采样（像素化效果）
  • crisp-edges - 保持锐利边缘（最近邻采样）
预期效果：pixelated 和 crisp-edges 应使用最近邻采样，auto 应使用双线性过滤。
<img> 元素测试
image-rendering: auto（默认）
image-rendering: auto;
应显示平滑的双线性过滤效果
image-rendering: pixelated
image-rendering: pixelated;
应显示像素化的最近邻采样效果
image-rendering: crisp-edges
image-rendering: crisp-edges;
应显示锐利边缘的最近邻采样效果
缩放效果测试
缩放 + auto（双线性）
width: 128px; height: 72px; transform: scale(2); image-rendering: auto;
缩放后应保持平滑效果
缩放 + pixelated（最近邻）
width: 128px; height: 72px; transform: scale(2); image-rendering: pixelated;
缩放后应显示明显的像素块效果
background-image 测试
background-image + auto
background-image: url("../images/bg.png"); image-rendering: auto;
背景图片应使用双线性过滤
background-image + pixelated
background-image: url("../images/bg.png"); image-rendering: pixelated;
背景图片应使用最近邻采样
