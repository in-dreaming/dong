CSS Logical Properties Test

This test verifies that CSS logical properties are correctly parsed and mapped to physical properties.

Current writing-mode: horizontal-tb, direction: ltr

Expected mapping: inline-start = left, inline-end = right, block-start = top, block-end = bottom

Test 1: Individual Logical Properties

margin-inline-start: 20px
Should have left margin of 20px
margin-inline-end: 30px
Should have right margin of 30px
margin-block-start: 15px
Should have top margin of 15px
margin-block-end: 25px
Should have bottom margin of 25px
padding-inline-start: 20px
Should have left padding of 20px
padding-inline-end: 30px
Should have right padding of 30px
padding-block-start: 15px
Should have top padding of 15px
padding-block-end: 25px
Should have bottom padding of 25px
border-inline-start: 4px solid red
Should have left border: 4px solid red
border-inline-end: 3px dashed blue
Should have right border: 3px dashed blue
border-block-start: 5px dotted green
Should have top border: 5px dotted green
border-block-end: 2px double purple
Should have bottom border: 2px double purple
inset-inline-start: 10px
Should have left offset of 10px
inset-inline-end: 20px
Should have right offset of 20px
inset-block-start: 5px
Should have top offset of 5px
inset-block-end: 15px
Should have bottom offset of 15px

Test 2: Shorthand Logical Properties

margin-inline: 20px 40px
Should have left margin 20px, right margin 40px
margin-block: 10px 30px
Should have top margin 10px, bottom margin 30px
padding-inline: 15px 25px
Should have left padding 15px, right padding 25px
padding-block: 5px 20px
Should have top padding 5px, bottom padding 20px
border-inline: 3px solid orange
Should have left and right borders: 3px solid orange
border-block: 2px dashed teal
Should have top and bottom borders: 2px dashed teal
inset-inline: 5px 15px
Should have left offset 5px, right offset 15px
inset-block: 10px 25px
Should have top offset 10px, bottom offset 25px

Test 3: Mixed Logical and Physical Properties

margin-left: 10px; margin-inline-start: 20px
padding-top: 5px; padding-block-start: 15px
border-right: 2px solid black; border-inline-end: 3px dashed red
Should show interaction between logical and physical properties

Test 4: Global Keywords with Logical Properties

margin-inline-start: initial
padding-block-end: inherit
border-inline: unset
Should handle global keywords correctly

Test Results Summary

If all tests pass, you should see:

  • Proper spacing and borders on all sides according to logical property values
  • Correct mapping of logical directions to physical directions (LTR horizontal-tb)
  • Shorthand properties working correctly with 1 or 2 values
  • No conflicts between logical and physical properties
  • Global keywords applied appropriately
