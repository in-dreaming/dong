Font-Weight Closest Match Test

Verifies that when requested weight variant is not available, the closest available weight is selected.

1. Non-100-Multiple Values (Normalization)

Values are normalized to nearest 100 based on standard CSS rules (>=50 rounds up, <50 rounds down)

Requested:150 150 AaBbCc 123 Should use 200 (Extra Light)
Requested:250 250 AaBbCc 123 Should use 300 (Light)
Requested:349 349 AaBbCc 123 Should use 300 (Light)
Requested:450 450 AaBbCc 123 Should use 500 (Medium)
Requested:550 550 AaBbCc 123 Should use 600 (Semi Bold)
Requested:650 650 AaBbCc 123 Should use 700 (Bold)
Requested:750 750 AaBbCc 123 Should use 800 (Extra Bold)
Requested:850 850 AaBbCc 123 Should use 900 (Black)

2. Outer Range Clamping

Values outside 100-900 range are clamped to nearest valid value

Requested:0 0 AaBbCc 123 Clamped to 100 (Thin)
Requested:50 50 AaBbCc 123 Clamped to 100 (Thin)
Requested:-100 -100 AaBbCc 123 Clamped to 100 (Thin)
Requested:950 950 AaBbCc 123 Clamped to 900 (Black)
Requested:1000 1000 AaBbCc 123 Clamped to 900 (Black)

4. Midpoint Weight Selection

When value is exactly between two weights (e.g., 450), rounds up

Requested:650 650 AaBbCc 123 Midpoint 600-700 → rounds to 700
