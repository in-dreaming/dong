Tabindex Sorting Test

Test Instructions:

Press Tab key repeatedly to navigate through focusable elements.

The focus order should follow the "Expected Focus Order" below.

Expected Focus Order (by tabindex):

  1. Input with tabindex="1" (lowest positive)
  2. Button with tabindex="2"
  3. Input with tabindex="5"
  4. Input with tabindex="10" (highest positive)
  5. Default input (no tabindex, defaults to 0)
  6. Button with tabindex="0"
  7. Default button (no tabindex, defaults to 0)
  8. Content editable div

Note: Element with tabindex="-1" is skipped in Tab navigation but can be focused programmatically.

tabindex="-1" - Skipped in Tab navigation
Content editable div (no tabindex attribute)
