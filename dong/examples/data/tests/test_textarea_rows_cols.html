Textarea rows/cols Attributes

Default (no rows/cols)
Should use: rows=2, cols=20 (defaults: height≈48px, width≈160px+padding)
rows="3" cols="10"
3 rows × 20px = 60px height
10 cols × 8px (0.5em) = 80px width
rows="5" cols="40"
5 rows × 20px = 100px height
40 cols × 8px (0.5em) = 320px width
rows="1" cols="30"
1 row × 20px = 20px height
30 cols × 8px (0.5em) = 240px width
Large textarea (rows="10" cols="60")
10 rows × 20px = 200px height
60 cols × 8px (0.5em) = 480px width
rows="15" cols="80"
15 rows × 20px = 300px height
80 cols × 8px (0.5em) = 640px width
